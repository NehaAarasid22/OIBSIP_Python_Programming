# BMI Calculator

## 📌 Project Description
This is a simple Python BMI Calculator that calculates a user's Body Mass Index (BMI) based on their weight and height. It also classifies the BMI into different health categories.

## 🚀 Features
- Takes weight and height as input
- Calculates BMI
- Displays BMI up to 2 decimal places
- Shows BMI category
- Handles invalid input
- Prevents negative or zero values

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run
1. Run the Python program.
2. Enter your weight in kilograms.
3. Enter your height in meters.
4. View your BMI and health category.

## 📊 BMI Categories
- Underweight : BMI < 18.5
- Normal Weight : BMI 18.5 - 24.9
- Overweight : BMI 25 - 29.9
- Obese : BMI ≥ 30
