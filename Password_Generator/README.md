# 🔐 Random Password Generator

## 📌 Project Description

This project is a simple Random Password Generator built using Python. It allows users to generate secure and random passwords by choosing the password length and selecting whether to include uppercase letters, numbers, and special symbols.

---

## 🚀 Features

- Generate random passwords
- Custom password length
- Include uppercase letters (optional)
- Include numbers (optional)
- Include special symbols (optional)
- Minimum password length validation
- Handles invalid numeric input using try-except

---

## 🛠️ Technologies Used

- Python 3
- random module
- string module

---

## ▶️ How to Run

1. Run the Python program.
2. Enter the desired password length (minimum 8 characters).
3. Choose whether to include:
   - Uppercase letters (y/n)
   - Numbers (y/n)
   - Special symbols (y/n)
4. The program will generate and display a random password.

---

## 📋 Example

### Input

```text
Enter password length: 10
Include uppercase letters? y
Include numbers? y
Include symbols? y
```

### Output

```text
Generated Password: A8@mK2#xLp
```

> Note: The generated password will be different every time because it is created randomly.
